HOOP UP

PLAYGROUNDS ARE PROVING GROUNDS

ABOUT

The program for every basketball player

HOOP UP, designed and coded by Shannon Lau, is a basketball tracking program that records and processes in-game statistics using a Python back-end. After creating an account, the user is able to input each game's statistics and ultimately view their progress in data charts powered by Google Charts. The program produces numbers and facts concerning your overall game and compares games side-by-side for in-depth analysis. With the use of this simplistically programmed software, the user can develop his or her game to become a better player. Know your game, know your grounds. Let's HOOP UP.
